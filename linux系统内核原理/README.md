## linux内核原理
```
系统api -> 语言 -> 语言的标准库 -> 各种三方库,框架,工具 -> 上层应用,业务代码,产品

一门语言,无论java,python,go,其实都是基于linux系统之上的开发工具.
语言封装了操作系统的api(系统调用),提供了自带的标准库,
框架又基于标准库封装了各种方便好用的工具,

开发这件事是
从下而上地写应用层/业务层,直接采用各种语言,框架,工具等,最快最高效地写出产品,投放市场,赚钱.

而学习技术,懂原理则是
从上往下地去拆解工具,框架,甚至拆解语言本身,直至系统api为止(硬件层了解即可),

所以,linux内核原理部分,cpu,mem,io调度等等的系统原理,才是真正能拉开水平差距的,深入的底层技术原理!

所谓“知其然,知其所以然”,才是真正的技术高手.
```
